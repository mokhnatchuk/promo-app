# Promo App

Сервіс для пошуку акцій та знижок на продукти в українських магазинах.

## Tech Stack

- **FE:** React, Bootstrap
- **BE:** Node.js, Express
- **DB:** SQLite
- **Auth:** JWT, bcryptjs
- **DevOps:** Docker

# Запуск-тест

Щоб запустити потрібно

В терміналі:

cd "f:/promo fullstack/promo-app/backend" && npm install && npm start
В іншому терміналі: cd "f:/promo fullstack/promo-app/frontend" && npm install && npm run dev
