# Promo App
Сервіс для пошуку акцій та знижок на продукти в українських магазинах.

## Tech Stack
- **FE:** React, Bootstrap
- **BE:** Node.js, Express
- **DB:** SQLite
- **Auth:** JWT, bcryptjs
- **DevOps:** Docker
